# 2.-Bank-Account-Balance-Checker-
2. Bank Account Balance Checker 
Objective:
To fetch and display the account balance of a user by entering an account number using JDBC.
Description:
In this project, the user enters an account number, and the system retrieves the corresponding balance from the database. It demonstrates the use of JDBC SELECT queries, ResultSet, and safe querying.
Requirements:
Java:
Console input using Scanner
JDBC connection setup
ResultSet for reading data
JDBC Concepts:
SELECT statement
PreparedStatement with parameters
ResultSet navigation
SQL Table:
accounts(acc_no INT PRIMARY KEY, holder_name VARCHAR(50), balance DOUBLE)
